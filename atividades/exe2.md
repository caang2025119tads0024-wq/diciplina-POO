#  Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?
quando o vetor é inicializado com tamanho fixo, o vetor tamanho precisa ser informado o para que ele possa reservar o espaço na memoria, ja quando ele é inicializado com valores especificos ele não precisa ser informado o o tamanho do vetor porque o compilador infere o tamanho automaticamente.
