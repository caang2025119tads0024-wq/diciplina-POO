#  Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?
quando o array é inicializado com tamanho fixo, o tamanho do vetor precisa ser informado para que ele possa reservar o espaço na memoria, ja quando ele é inicializado com valores especificos ele não precisa ser informado o tamnho do vetor porque o compilador infere o tamanho automaticamente
